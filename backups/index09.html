Jeremy Ellis / webmcu-ai test v09: sounds now fire on a steady tempo clock instead of the instant a position changes. Your body position still decides WHAT plays (fully continuous, unchanged), but WHEN it plays is now locked to the beat below. Legs replay their current drum every tick while held in a zone (like a drum machine). Arms only re-trigger on a tick if the note actually changed since the last tick (so a held note doesn't machine-gun-repeat). Angle readings are also lightly smoothed now, which should reduce one leg falsely triggering from the other leg's movement.

ARMS - fold across chest = silence, no other silence/gaps once extended - every angle plays some note. 0 degrees (straight up) is always the physical note C. The dropdown above only changes which notes are available and how much angular room each one gets - it does NOT move where C sits.
